# 💰 Budget Tracker App

A simple Android application that helps users track their income and expenses efficiently. The app allows users to add, view, and manage financial transactions in an easy-to-use interface.

---

## 📱 Features

- ➕ Add income and expenses
- 📋 View all transactions in a list
- 💾 Local data storage using SQLite
- 📊 Basic budget tracking (income vs expenses)
- 🧹 Delete or manage entries
- 🎨 Simple and clean user interface

---

## 🛠️ Built With

- Java (Android Development)
- Android Studio
- XML (UI Design)
- SQLite (Local Database)
- RecyclerView (Data Display)
